#### 2. 优化城市搜索匹配规则
**问题描述**：
- 城市搜索使用严格匹配，无法搜索 `长沙·岳麓·麓谷` 等城市
- 用户输入 `长沙` 无法匹配完整城市名称

**解决方案**：
- 修改 `backend/apps/jobs/views.py`
- 将 `city=city` 改为 `city__icontains=city`
- 支持模糊匹配，不区分大小写

**修改内容**：
```python
# 修改前
if city:
    queryset = queryset.filter(city=city)  # 严格匹配

# 修改后
if city:
    queryset = queryset.filter(city__icontains=city)  # 模糊匹配
```
